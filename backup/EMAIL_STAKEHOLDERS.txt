TWO USER ROLES:

ADMIN (Full Access):
  TAB 1: CONTACT SUBMISSIONS
  • View all contact form submissions in one place
  • Real-time statistics: Total leads, new today, status breakdown
  • Filter by status: New, Contacted, Converted, Closed
  • Search by name, email, phone, or city
  • Update lead status and add internal notes
  • Track which leads came from where

  TAB 2: DOWNLOADS MANAGEMENT
  • Upload files (PDFs, presentations, documents, etc.)
  • Add title and description for each file
  • Toggle visibility on/off without deleting
  • See download statistics for each file
  • Delete files when no longer needed

  TAB 3: SETTINGS
  • Change own password anytime
  • Secure password update with verification

PORTAL MANAGER (Downloads Only):
  • Upload marketing materials (brochures, presentations, fact sheets)
  • Manage file visibility and descriptions
  • View download statistics
  • Delete outdated materials
  • Change own password
  • ✗ Cannot access customer contact data
  • ✗ Cannot see lead submissions
